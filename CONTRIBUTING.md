# Contributing

This is a source-free public distribution repository. Contributions are limited
to public docs, installer behavior, issue templates, and release metadata.

Do not submit:

- Vixy source code
- Reverse-engineered code
- Private build scripts
- Secrets or customer data
- Binary release artifacts

Before opening a pull request:

1. Run `bash -n install.sh`.
2. Keep examples free of private paths and credentials.
3. Update `CHANGELOG.md` when public distribution behavior changes.
